처음으로 만드는 웹 페이지

처음으로 만드는 HTML 문서로 작성된 Web Page, Web Page는 HTML 형식의 문서로 작성한다 HTML p Tag로 시작하면 하나의 문단이 시작된다. 하나의 문단은 close /p 이며 문단이 끝났음을 나타낸다. 문단내에서는 아무리 Enter를 입력하여도 Browser에 나타날때는 모두 무시된다. 단 Enter를 입력시 한칸의 빈칸이 나타난다. 문단 중간에서 여러개의 빈칸(Space)를 입력하여도 한개의 빈칸만 남기고 모두 무시된다.

HTML 문서는 tag로 감싸진 Text로 구성되어 있다 Tag로 감싸긴 문서를 작성하고 Web Browser를 통하여 확인한다
